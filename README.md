# 注册机特色
1. 真实浏览器指纹注册
2. 自动打码
3. 手动纠正打码
4. 自动生成用户名
5. 自动生成姓名

# 注意事项
**需要干净的ip和可用的邮箱，通过率会大幅提高！**

# 安装依赖，进入到源码目录，这一步会自动安装需要的依赖
```
pip install --break-system-packages -r requirements.txt
```

# 执行 程序
```
python3 main.py
```
